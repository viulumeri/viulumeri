Android-ohjeet:

1. Avaa sovellus selaimessa
2. Asenna sovellus:
    - Automaattisen promptin kautta: jos selain tarjoaa asennusta automaattisesti, paina "Asenna"
    - Manuaalisesti: paina selaimen valikkoa (⋮) ja valitse "Lisää aloitusnäytölle"
3. Paina "Asenna"

iPhone / iPad ohjeet:
1. Paina selaimen kolmepistevalikkoa näytön alareunassa
2. Paina Share / Jaa -painiketta (neliö, josta lähtee nuoli ylöspäin)
3. Valitse "Add to Home Screen" / "Lisää aloitusnäyttöön"
